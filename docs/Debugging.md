# Logging, tracing, and debugging

## API Logging

From the teletype or the HTML console issue the following command:  
`;log <module> <level>`  

For example, to enable the most verbose logging on the AUDIO api:  
`;log audio all`  

To disable logging:  
`;log audio none`  

### Log levels
 - NONE
 - CRITICAL
 - ERROR
 - WARNING
 - INFO
 - ALL 

### API modules:
- AUDIO
- AUTOMATON
- GDB
- MACHINE
- NETWORK
- TELETYPE
- VIEWS

## Teletype cheat commands

- ;Trace
