# Resolution Categories

Low: 1280×720 (HD).  
Medium: 1920×1080 (FHD).  
High: 2560×1440 (QHD).  
Ultra High: 3840×2160 (4K).  

Aspect Ratios

16:9 (most common for displays)
16:10 (common for laptops)
4:3 (older tablets, some displays)
19:9/20:9 (modern smartphones)
3:2 (Surface devices, some laptops)

Breakpoints (width in logical pixels)

Small: 320-480px (small phones)
Medium: 481-768px (large phones, small tablets)
Large: 769-1024px (tablets, small laptops)
X-Large: 1025-1440px (laptops, desktops)
XX-Large: 1441px+ (large desktops)

Orientations

Portrait
Landscape

Density-Independent Image Assets
Create image assets at these scales to handle various display densities:

1x (baseline)
2x (Retina displays, high-end Android)
3x (iPhone Pro models, high-end displays)
