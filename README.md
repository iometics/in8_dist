# in8 Framework - Xcode Project Setup Documentation

This repository contains documentation and guides for setting up Xcode projects (macOS and iOS) using the in8 framework - an SDL3-based cross-platform game framework.

## 📚 Documentation

- **[Xcode Project Setup Guide](xcode_project_setup_guide.md)** - Complete step-by-step guide for creating macOS and iOS projects
- **[C++ Main Files](cpp_main_files.md)** - Minimal C++ entry points for both platforms

## 🎯 What This Repository Is For

This documentation helps developers:
- Create minimal Xcode projects that integrate the in8 framework
- Understand the basic setup process for macOS and iOS platforms
- Use C++, Objective-C, or Swift as the wrapper language
- Structure their projects similar to the Android Studio setup

## 📋 Prerequisites

To use this documentation, you need:
- Xcode 14.0 or later
- macOS development machine
- **The in8.xcframework** (see Obtaining the Framework below)
- Basic familiarity with Xcode

## 🏗️ What You'll Build

These guides help you create minimal wrapper projects that:
- Contain only a simple main file (C++, Objective-C, or Swift)
- Link against the in8.xcframework
- Launch your SDL3-based game
- Mirror the simplicity of the Android Studio setup

## 📱 Supported Platforms

The in8 framework and these guides support:
- ✅ macOS (Intel and Apple Silicon)
- ✅ iOS (Device and Simulator)
- ✅ iPadOS
- ✅ Android (via separate Android Studio project)
- ✅ Linux (via separate build system)
